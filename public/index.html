API DOKUMENTASI WISATA INDONESIA DEV

API Wisata Indonesia Dev menyediakan data lengkap mengenai provinsi Indonesia dan rekomendasi wisata di dalamnya dengan total 999+ wisata.

URL : Wisata Indonesia Dev

Api ini berisi data seluruh wilayah indonesia mulai dari :

  • Provinsi
  • Rekomendasi Wisata

Wisata Berisi data seperti :

  • Nama
  • Alamat
  • Gambar
  • Link
  • Rating
  • Dan lainnya
